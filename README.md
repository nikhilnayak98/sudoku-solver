# Sudoku Solver
Sudoku solver in python.

<code>python3 main.py</code>

- row - r
- column - c
- square - s
- digit - d
- unit - u, ['A1','B1','C1','D1','E1','F1','G1','H1','I1']
- grid - grid, 81 non-blank chars, starting with '.18...7...
- values - dict of possible values, {'A1':'12349', 'A2':'8', ...}
